# AI日報・経営管理システム　市場調査資料（統合版）

---

## 1. サービス概要

### システム名（仮）
**AI経営クラウド** ― LINEで送るだけ、日報から経営管理まで全自動

### キャッチコピー
> 「日報を送るだけで、請求書も損益もリアルタイムでわかる」

### サービス概要
電気工事・建設業・設備工事などの小規模事業所向けに、LINEを使った日報入力から請求集計・経費管理・損益計算・経営レポートまでを全自動化するクラウドサービス。

**2つのシステムが連動して動作：**

| システム | 役割 |
|----------|------|
| 日報システム | 現場スタッフが日報をLINEで送信→稼働・請求を自動集計 |
| 経営管理システム | 経費・売上・外注を管理→案件別損益・月次レポートを自動生成 |

## 5. 機能一覧

### 【日報システム】現場スタッフ向け
- LINEで自然文を送信するだけで入力完了
- AI（Claude）による自然言語解析・略称自動変換
- OneDrive Excelへの自動書き込み
- 重複送信の自動検出・エラー通知
- 複数現場・複数人員の一括入力対応

### 【日報システム】請求集計
- 日報送信と同時に請求集計Excelへ自動反映
- 同日・同現場の複数人員を自動集約
- 20日締め・末締めの自動判定
- 経費（高速・駐車場・資材費）の自動合算
- 元請け会社別シートで管理

### 【経営管理システム】経費・外注・売上管理
| 機能 | LINEコマンド例 |
|------|--------------|
| 経費登録 | 「経費　交通費　500円　コンビニ」 |
| レシート登録 | レシートの写真を送るだけ（AI自動読取） |
| 外注請求登録 | 「外注請求　〇〇社　△△工事　50,000円」 |
| 売上登録 | 「売上　〇〇社　△△工事　100,000円」 |
| 入金確認登録 | 「入金　〇〇社　110,000円」 |
| 外注支払登録 | 「外注支払　〇〇社　55,000円」 |

### 【経営管理システム】照会・確認
| コマンド | 内容 |
|----------|------|
| 未払い確認 | 外注の未払い一覧をLINEで即表示 |
| 未入金確認 | 売上の未入金一覧をLINEで即表示 |
| 今月サマリー | 売上・入金・外注・経費の集計 |
| 今月の粗利は？ | 粗利をLINEで確認 |
| 支払期限確認 | 期限が近い支払い一覧 |

### 【連携機能】日報 × 経営管理
- 日報の移動費（高速・駐車場）を経費として自動連携
- 案件別の損益計算（売上 − 外注費 − 人件費 − 移動費）
- 月次レポートに人員別稼働時間・人件費を自動追加

### 【自動スケジュール機能】
| タイミング | 内容 |
|-----------|------|
| 毎月25日 | 締日リマインダー（外注未払い・売上未入金一覧）をLINE通知 |
| 毎月1日 | 前月の月次経営レポートをLINE送信 |
| 毎日 | 入金期限超過アラートを自動通知 |

## 11. 想定Q&A

**Q: LINEを使うのはセキュリティ的に大丈夫ですか？**
A: データはOneDrive（Microsoft）に保存されるため企業レベルのセキュリティです。

**Q: Excelのスキルが必要ですか？**
A: 不要です。入力はLINEだけで完結します。管理者はExcelを閲覧するだけです。

**Q: 経費のレシートはどうやって登録するのですか？**
A: LINEでレシートの写真を送るだけです。AIが金額・店名・種類を自動で読み取ります。

**Q: 案件ごとの損益はどこで確認できますか？**
A: LINEで「△△工事の損益」と送るだけで、売上・経費・人件費・実粗利が表示されます。

**Q: 既存のExcelフォーマットは使えますか？**
A: カスタマイズ対応可能です。

**Q: インターネット環境がない現場でも使えますか？**
A: LINEが使える環境（4G/5G含む）があれば使用可能です。
