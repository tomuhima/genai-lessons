# 経営管理システム 引き継ぎ資料

## 概要
- システム名：経営管理システム
- インフラ：n8n 2.14.2（Self Hosted / XServer VPS / Docker）
- UI：LINE公式アカウント（日報システムとは別チャンネル）
- ストレージ：OneDrive `/経費管理/経費管理_YYYY年MM月.xlsx`
- ワークフローファイル：workflows/business_management_workflow_v2.json

## 入力機能（LINEで送信）

### 経費
```
経費
種類：交通費
店名：コンビニ
金額：500
支払方法：現金
```
記録項目：日付・種類・支払方法・店名・案件・金額・備考

### 外注請求
```
外注請求
業者：〇〇社
案件：△△工事
金額：50000
支払期限：2026/05/20
```
記録項目：登録日・業者・案件・税抜金額・消費税・合計・請求日・支払期限・支払状況

### 外注支払
```
外注支払
業者：〇〇社
金額：55000
支払日：2026/04/25
```
記録項目：業者・支払日・支払金額（外注管理シートの支払状況を「未払」→「支払済」に更新）

### 売上
```
売上
得意先：〇〇社
案件：△△工事
金額：100000
支払期限：2026/05/31
```
記録項目：登録日・得意先・案件・税抜金額・消費税・合計・請求日・入金期限・入金状況

### 入金
```
入金
得意先：〇〇社
金額：110000
入金日：2026/04/20
```
記録項目：得意先・入金日・入金額（売上管理シートの入金状況を「未入金」→「入金済」に更新）

### レシート写真
- LINE上で写真を送信
- Claude APIが画像を読み取り自動で経費データを抽出・記録

---

## 照会機能（LINEで確認）

| コマンド | 内容 |
|----------|------|
| ヘルプ | 使い方一覧を表示 |
| 未払い確認 | 外注の未払い一覧 |
| 〇〇社の未払い | 特定業者の未払い |
| 未入金確認 | 売上の未入金一覧 |
| 今月サマリー | 売上・入金・外注・経費の集計 |
| 今月の粗利は？ | 粗利確認 |
| 今月の経費合計は？ | 経費合計確認 |
| 支払期限確認 | 期限が近い支払一覧 |

---

## 自動機能（スケジュール実行）

| 機能 | タイミング | 内容 | 通知先 |
|------|-----------|------|--------|
| 締日リマインダー | 毎月25日 10:00 | 外注未払い・売上未入金の一覧をLINE送信 | 野添優・野添アキ子 |
| 月次レポート | 毎月1日 09:00 | 前月の売上・入金・外注・経費の集計レポート | 野添優・野添アキ子 |
| 入金遅延チェック | 毎日 09:00 | 入金期限を過ぎた未入金があればアラート | 野添優・野添アキ子 |
