# AI日報・経営管理システム　市場調査資料

---

## 1. サービス概要

### システム名（仮）
**AI日報クラウド** ― LINEで送るだけ、全自動で経営管理

### キャッチコピー
> 「日報を送るだけで、請求書まで自動でできる」

### サービス概要
電気工事・建設業・設備工事などの小規模事業所向けに、
LINEを使った日報入力からExcel集計・請求管理・経営レポートまでを
全自動化するクラウドサービス。

## 3. 現状の業界課題

### 小規模事業所が抱える問題
| 課題 | 詳細 |
|------|------|
| 日報管理が手作業 | 紙・LINEグループ・メールなどバラバラ |
| 集計に時間がかかる | 月末に経理担当者が手作業で集計 |
| 請求漏れが発生する | 現場記録と請求書の突き合わせが不正確 |
| ITツールが使いこなせない | スタッフがスマホ以外の操作に不慣れ |
| 専用システムが高い | 既存の建設管理システムは月額数万円〜 |

## 5. 機能一覧

### 日報管理
- LINEで自然文を送信するだけで入力完了
- AI（Claude）による自然言語解析・略称自動変換
- OneDrive Excelへの自動書き込み
- 重複送信の自動検出・エラー通知
- 複数現場の一括入力対応
- 複数人員の記録対応

### 請求集計
- 日報送信と同時に請求集計Excelへ自動反映
- 同日・同現場の複数人員を自動集約
- 20日締め・末締めの自動判定
- 経費（高速・駐車場・資材費）の自動合算
- 会社別シートで管理

### 経営管理
- 経費・売上のLINE入力
- 月次レポートの自動生成・LINE送信
- 締日リマインダーの自動通知
- 入金遅延アラートの自動通知

### 管理者機能
- LINEで請求集計を即時確認
- 全社員の稼働状況を一括確認
- 月次収支レポートの自動受信

## 10. 想定Q&A

**Q: LINEを使うのはセキュリティ的に大丈夫ですか？**
A: データはOneDrive（Microsoft）に保存されるため企業レベルのセキュリティです。

**Q: Excelのスキルが必要ですか？**
A: 不要です。入力はLINEだけで完結します。管理者はExcelを閲覧するだけです。

**Q: 既存のExcelフォーマットは使えますか？**
A: カスタマイズ対応可能です。

**Q: インターネット環境がない現場でも使えますか？**
A: LINEが使える環境（4G/5G含む）があれば使用可能です。
