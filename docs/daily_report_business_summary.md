# 日報システム ビジネス展開 引き継ぎ資料

## 作成者・背景
- 小規模設備系事業者が自社業務効率化のために自作
- 「既存サービスでここまでできるものがない」という課題から開発
- Claude Code + n8n で構築

---

## 現在のシステム構成

### インフラ
- 自動化：n8n 2.14.2（Self Hosted / XServer VPS / Docker）
- UI：LINE公式アカウント
- AI解析：Claude API（claude-sonnet-4-6）
- ストレージ：OneDrive（Microsoft 365）
- n8n URL：https://n8n-light-mn.xvps.jp

### 機能① 日報受付・記録
- LINEにテキストで日報送信
- Claude APIが自然文から項目を自動抽出
- OneDrive `/業務管理システム/日報/稼働表_YYYY年MM月.xlsx` に保存
- 記録項目：日付・現場名・区分・開始/終了時刻・休憩・作業内容・高速・距離・駐車場など
- 完了時に送信者へLINE返信

### 機能② 請求集計自動化
- 日報受付と同時に自動実行
- 請求対象会社：港振興業・トラストテクノス・平成システム・FGE・mtr・オークコミュニケーション・ライズ・千里スカイハイツ管理組合・菊次・ページ・G-STYLE・梶原通信
- OneDrive `/業務管理システム/請求集計/請求集計_YYYY年MM月.xlsx` に保存
- 記録項目：日付・区分・現場名・人員名・高速(円)・距離(km)・駐車場(円)
- 締め日：平成システムのみ20日締め（前月21日〜当月20日）、他は月末締め
- 会社ごとにシートを分けて追記
- 完了時に野添優へLINEプッシュ通知

## ビジネス展開の検討内容

### ターゲット市場
- 従業員10人以下の小規模設備・建設・警備会社
- 日報・請求集計をExcelで手作業している会社
- ITツール導入コストを抑えたい会社

### 差別化ポイント
- LINEで完結（専用アプリ不要・導入ハードル低い）
- 中小零細向け（現場作業員でも使える）
- 柔軟なカスタマイズ（締め日・請求先対応）
- 低コスト（既存のLINE・OneDriveを活用）

### 競合
- ジョブカン・freee・マネーフォワード・Kintone・蔵衛門など
- 大企業向けが多く、小規模向けは手薄

### 収益モデル案
- 月額3,000〜5,000円/社
- まず知り合い2〜3社に無料提供 → フィードバック改善 → 有償化

### 次のステップ案
1. PWA化（Webアプリ）で入力UIを改善
2. 同業他社2〜3社に無料で使ってもらう
3. 月額有償化
4. 口コミ・紹介で拡大
