# 日報システム × 経営管理システム 連携設計メモ

作成日：2026-04-25
更新日：2026-04-25

---

## 現状の問題点

両システムは現在「並列稼働」であり「連携」していない。
データが繋がっていないため、案件単位の損益（実粗利）が計算できない。

```
【現状】
日報システム  → 稼働表Excel（/日報/）
経営管理システム → 経費管理Excel（/経費管理/）

← この2つは別ファイル・別シートで完全に独立している →
```

---

## 連携で目指すデータフロー

```
LINEで日報送信
    ↓
稼働表に記録（人員・現場・時間・移動費）
    ↓
    ├─→ 請求集計（得意先別）
    └─→ 経営管理システムへ自動連携
            ↓
     案件別損益計算
      売上
    - 自社人件費（稼働時間 × 日当）  ← 日報から自動取得
    - 外注費                          ← 経営管理から取得
    - 移動経費（高速・駐車場）        ← 日報から自動取得
    - その他経費                      ← 経営管理から取得
    = 実粗利
```

## 経営管理システムに追加すべき項目

### ① 案件マスタ（最優先）

**なぜ必要か：**
日報の「現場名」と経営管理の「案件名」が自由記述のため、
同じ現場が別名で登録されてデータが分断される。
共通の案件IDで両システムを紐付ける必要がある。

**追加すべき内容：**

| 項目 | 内容 |
|------|------|
| 案件ID | 例：2026-001（年-連番） |
| 案件名 | 正式名称（両システムで統一） |
| 得意先名 | 請求先（日報の請求集計と一致させる） |
| 契約金額（税抜） | 見積・契約ベース |
| 開始日・完了予定日 | 工期管理用 |
| ステータス | 進行中 / 完了 / 請求済 / 入金済 |

**LINEコマンド例（追加）：**
```
案件登録
案件名：△△ビル電気工事
得意先：〇〇社
金額：500000
開始：2026/05/01
```

### ② 移動費の自動連携

**対応方針：**
- 日報保存時に移動費（高速・駐車場）を経費管理Excelの「経費」シートへ自動追記
- 距離（km）は後で燃料費計算に使えるよう保持する
- 案件IDで紐付けることで案件別移動コストが自動集計できる

## 設計上の注意点

### 案件名の表記統一が最重要
日報の「現場名」と経営管理の「案件名」が少しでも違うと連携できない。
**案件IDを先に決めて、両システムでIDを使う運用にする**のが安全。

### 外注扱い人員の処理
服部秀一・梶原通信・井本・RISE・トラストテクノスは全て外注。
稼働時間は日報に記録されるが、人件費計算には含めず外注費として管理する。

### 野添アキ子の扱い
専従事務員のため日報送信なし。固定給100,000円を月次固定費として記録。案件別損益には含めない。

### 月次ファイル設計
案件は月をまたぐため、案件損益だけは「年間ファイル」1本にまとめることを推奨。
