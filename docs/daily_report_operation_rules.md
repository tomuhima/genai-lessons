# 日報送信 運用ルール

---

## 送信先
LINEの日報用公式アカウントにメッセージを送信してください。

---

## 基本フォーマット

### 一般作業員（野添優・丸田翔吾・源地健史・服部秀一）

```
日報
日付 区分 会社名 物件名 開始〜終了
```

### 梶原通信

```
日報
日付 区分 会社名 物件名 人員：名前 名前 名前 開始〜終了
```

---

## 記入例

### 一般作業員

**1現場（経費なし）**
```
日報
4/25 昼 平成 灘駅 8〜17
```

**1現場（経費あり）**
```
日報
4/25 昼 平成 灘駅 8〜17 高速2920 距離76 駐車800
```

**夜勤**
```
日報
4/25 夜 平成 灘駅 20〜翌1
```

**複数現場**
```
日報 4/25

【現場①】
昼 平成 灘駅 8〜14

【現場②】
昼 コウシン ○○マンション 14:30〜18 駐車800
```

---

### 梶原通信

**1現場（経費なし）**
```
日報
4/25 昼 平成 灘駅 人員：梶原 今川 前原 8〜17
```

**1現場（経費あり）**
```
日報
4/25 昼 平成 灘駅 人員：梶原 今川 前原 8〜17 高速2920 距離76 駐車800
```

**複数現場**
```
日報 4/25

【現場①】
昼 平成 灘駅 人員：梶原 今川 8〜14 高速2920 距離45

【現場②】
昼 コウシン ○○マンション 人員：梶原 14:30〜18
```

| 項目 | 一般作業員 | 梶原通信 | 省略時 |
|------|-----------|---------|--------|
| 日付 | **必須** | **必須** | - |
| 区分（昼・夜） | **必須** | **必須** | - |
| 会社名 | **必須** | **必須** | - |
| 物件名 | **必須** | **必須** | - |
| 人員 | 不要 | **必須** | - |
| 開始〜終了 | **必須** | **必須** | - |
| 高速料金 | 省略可 | 省略可 | 0 |
| 移動距離 | 省略可 | 省略可 | 0 |
| 駐車場 | 省略可 | 省略可 | 0 |

| 休憩 | 省略可 | 省略可 | 0 |
| 作業内容 | 省略可 | 省略可 | 空白 |

---

## 注意事項

- 1行目は必ず「日報」から始めてください
- 複数現場は必ず【現場①】【現場②】で区切ってください
- 夜勤で日付をまたぐ場合は**開始日の日付**を入力してください
  - 例）4/25 21:00〜4/26 02:00 の場合 → 日付：4/25
- 梶原通信の人員は（n名）の記載不要。名前の数から自動カウントします
