# AI経営管理システム　市場調査資料

---

## 1. サービス概要

### システム名（仮）
**AI経営クラウド** ― LINEで送るだけ、経費から損益まで全自動管理

### キャッチコピー
> 「レシートを撮るだけで、月次決算が自動でできる」

### サービス概要
電気工事・建設業・設備工事などの小規模事業所向けに、
LINEを使った経費・売上・外注の管理から月次レポート・入金管理まで
全自動化するクラウドサービス。

**経理の専門知識不要。スマホのLINEだけで経営数字が把握できる。**

## 3. 現状の業界課題

| 課題 | 詳細 |
|------|------|
| レシート管理が手作業 | 紙のレシートを袋に入れて月末にまとめて入力 |
| 外注支払いの管理が煩雑 | 誰にいくら払ったか・未払いがあるかわからない |
| 売上・入金の管理が曖昧 | 請求したが入金されていない案件を見落とす |
| 月次の経営数字が把握できない | 結局いくら儲かったのかが月末にならないとわからない |
| 会計ソフトが使いこなせない | 弥生・freeeなどの操作が難しく続かない |
| 外出中に確認できない | 事務所に戻らないと帳簿が見られない |

## 5. 機能一覧

### 経費登録
| 方法 | 内容 |
|------|------|
| テキスト入力 | 「経費　交通費　500円　コンビニ」と送信するだけ |
| レシート写真 | 写真を送るだけ → AI（Claude）が金額・店名・種類を自動読取 |

**記録項目：** 日付・種類・支払方法・店名・案件・金額・備考

---

### 外注管理
**外注請求の登録：**
```
外注請求
業者：〇〇社
案件：△△工事
金額：50,000
支払期限：2026/05/20
```

**外注支払の登録：**
```
外注支払
業者：〇〇社
金額：55,000
支払日：2026/04/25
```
→ 支払済みステータスへ自動更新

---

### 売上・入金管理
**売上の登録：**
```
売上
得意先：〇〇社
案件：△△工事
金額：100,000
支払期限：2026/05/31
```

**入金の登録：**
```
入金
得意先：〇〇社
金額：110,000
入金日：2026/04/20
```
→ 入金済みステータスへ自動更新

---

### LINEで即時照会
| コマンド | 内容 |
|----------|------|
| 未払い確認 | 外注の未払い一覧を即表示 |
| 〇〇社の未払い | 特定業者の未払いを確認 |
| 未入金確認 | 売上の未入金一覧を即表示 |
| 今月サマリー | 売上・入金・外注・経費の集計を表示 |
| 今月の粗利は？ | 粗利をLINEで即確認 |
| 今月の経費合計は？ | 経費合計を確認 |
| 支払期限確認 | 期限が近い支払い一覧を表示 |

## 11. 想定Q&A

**Q: 会計ソフト（freee・弥生）の代わりになりますか？**
A: 確定申告用の仕訳入力には対応していません。経費の収集・集計・損益の把握に特化しています。既存の会計ソフトと併用することを推奨します。

**Q: レシートの読み取り精度はどのくらいですか？**
A: AI（Claude）による読み取りのため、鮮明な写真であれば高い精度で読み取れます。読み取り後に確認・修正も可能です。

**Q: LINEを使うのはセキュリティ的に大丈夫ですか？**
A: データはOneDrive（Microsoft）に保存されるため企業レベルのセキュリティです。

**Q: 複数人（経理担当者と社長など）で使えますか？**
A: 複数のLINE IDに通知・権限を設定できます。

**Q: 日報システムとセットで導入できますか？**
A: 可能です。日報システムと連携することで移動費の二重入力がなくなり、案件別の損益計算が自動でできます。

**Q: インターネット環境がない現場でも使えますか？**
A: LINEが使える環境（4G/5G含む）があれば使用可能です。
