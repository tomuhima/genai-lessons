## 入力ルール

| 項目 | 内容 |
|------|------|
| 日付 | YYYY/MM/DD形式（例：2026/04/17） |
| 区分 | 昼勤 / 夜勤 / 半日（午前）/ 半日（午後） |
| 会社名 | 得意先名を正確に |
| 物件名 | 現場名を入力 |
| 人員 | 名前を「・」区切り、最後に（人数）を付ける |
| 開始・終了 | HH:MM形式（例：09:00） |
| 休憩 | 分単位で入力（例：60）。なければ「0」 |
| 高速料金 | 円単位。なければ「0」 |
| 移動距離 | km単位。なければ「0」 |
| 駐車場 | 円単位。なければ「0」 |
| 資材費 | 円単位。なければ「0」 |
| 複数現場 | 【現場①】【現場②】と番号を付けて区切る |

### 人員の書き方

```
人員：梶原・田中・鈴木（3名）    ← 3人の場合
人員：梶原・田中（2名）          ← 2人の場合
人員：梶原（1名）                ← 1人の場合
```

### 注意事項

- 1行目は必ず「日報」から始めてください
- 複数現場は必ず【現場①】から始めてください
- 経費が0円の場合も必ず「0」と入力してください
- 夜勤で日付をまたぐ場合は**開始日の日付**を入力してください
  - 例）4/17 21:00 〜 4/18 02:00 の場合 → 日付：2026/04/17
